## Full Stack boilerplate Authentication with JWT strategy.
### Tech Stack: 
### ● NodeJS with TypeScript in back-end.
### ● VueJS with JavaScript in front-end.
### Features:
#### ● Register Page.
#### ● Log in Page.
#### ● Log out Page.
#### ● Profile Page.
#### ● Dashboard Page.
#### ● Custom 404 Page.
#### ● Vue Route Guards.
#### ● Private / Protected Routes.
#### ● User information stored persisted in Pinia.
#### ● Responsive Design with Tailwind CSS.
#### ● Modern Notifications.
#### ● Error Handler.
#### ● Input Validation.
#### ● JWT authentication as an HTTP only cookie in browser. (Safer method)



### Setup locally:

### Server :

```
cp .env.example .env
```
```
npm install
```
```
npm run dev
```
### Client :
```
cd client
```
```
npm install
```
```
npm run dev
```
